# Darna Backend (NestJS)

## 1. Présentation du projet

### 1.1. Contexte
Le backend de Darna est conçu pour supporter l'application mobile et web dédiée à la recherche de colocation pour étudiants. Il assure la gestion des utilisateurs, des annonces, des communications et des paiements, tout en garantissant sécurité, performance et fiabilité.

### 1.2. Objectifs
- Fournir une API REST sécurisée pour le frontend (mobile et web).  
- Gérer les utilisateurs (étudiants, propriétaires, sponsors, agences).  
- Gérer les annonces de colocation, y compris la création, modification et suppression.  
- Supporter la messagerie interne et les demandes des utilisateurs.  
- Gérer les publicités et promotions sponsorisées.  
- Assurer la sécurité et la fiabilité des données.

## 2. Technologies utilisées

- **Framework** : NestJS (Node.js)  
- **Langage** : TypeScript  
- **Base de données** : MongoDB  
- **ORM / ODM** : Mongoose  
- **Authentification** : JWT (JSON Web Token)  
- **Gestion des requêtes HTTP** : Axios / HTTP module intégré  
- **Documentation API** : Swagger
